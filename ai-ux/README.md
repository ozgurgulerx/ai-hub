## Key Themes

| Theme | Core Insight |
|-------|--------------|
| **Set Expectations** | Users need to know what AI can and cannot do |
| **Make Quality Visible** | Trust through transparency (citations, controls, feedback) |
| **Handle Failures** | Graceful degradation and helpful recovery |
| **Keep Control** | Especially critical for autonomous agents |
| **Close the Loop** | User signals improve the system |
