## Key Concepts

| Concept | Description |
|---------|-------------|
| **Chunking** | Breaking documents into retrievable units |
| **Embeddings** | Dense vector representations for semantic search |
| **Hybrid Search** | Combining sparse (BM25) and dense retrieval |
| **Reranking** | Second-stage scoring for precision |
| **GraphRAG** | Graph-augmented retrieval for multi-hop reasoning |
| **Context Engineering** | Managing what goes into the LLM context window |
