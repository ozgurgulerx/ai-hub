# Causal ML

**Causal inference** and **causal machine learning**: estimating intervention effects, counterfactuals, and avoiding observational data pitfalls.

## What This Covers

- **Causal fundamentals**: potential outcomes, counterfactuals, identifiability
- **Graphs**: DAGs, backdoor/frontdoor, d-separation, do-calculus
- **Estimation**: matching, IPW, doubly robust estimators
- **Causal ML**: uplift modeling, CATE/HTE, causal forests, meta-learners
- **Experimentation**: A/B testing, sequential testing, interference
- **Failure modes**: selection bias, unmeasured confounding, collider bias
