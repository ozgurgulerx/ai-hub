# DeepSeek V3

Strong general-purpose base LLM used as foundation for reasoning-focused post-training.

## Key Points

- Base model for DeepSeek-R1 reasoning line
- Large-scale RL with outcome-based rewards
- Claims: 128K context, dynamic sparse attention
- MoE architecture with GQA/MLA
