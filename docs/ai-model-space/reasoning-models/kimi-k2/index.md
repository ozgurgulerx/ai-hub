# Kimi K2

Moonshot AI's reasoning-capable model for multi-step tasks and long-context use cases.

## Key Features

- **Long-context orientation** — strong on multi-doc synthesis, codebase navigation
- **Chinese + English** — bilingual strength
- **Agent readiness** — pairs well with tool use

## Architecture Notes

- KDA (Kimi Diagonal Attention) — per-dimension gating for selective forgetting
- Linear attention variant with learned decay rates
