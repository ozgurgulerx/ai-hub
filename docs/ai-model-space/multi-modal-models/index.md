# Multi-Modal Models

AI systems that operate over **multiple modalities** (text, images, audio, video, structured data).

## What This Covers

- **Modalities + representations**: text tokens, vision embeddings, audio features
- **Fusion patterns**: early fusion, late fusion, cross-attention, two-tower models
- **Multimodal retrieval (RAG)**: indexing strategies, video events/highlights
- **Evaluation**: modality-specific metrics, failure taxonomy
- **Safety**: prompt injection via multimodal content, PII in images/audio
